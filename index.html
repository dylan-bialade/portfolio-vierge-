SLAM

Mémento SLAM

C#

  • Déclarations et Types :
    • int x = 10; : Variable entière avec une valeur de 10.
    • string name = "John"; : Chaîne de caractères contenant "John".
    • bool isActive = true; : Variable booléenne avec valeur true (vrai).
    • double pi = 3.14159; : Variable à virgule flottante.
    • var age = 25; : Inférence de type basée sur la valeur fournie.
  • Dates et Heures :
    • DateTime now = DateTime.Now; : Date et heure actuelles.
    • DateTime specificDate = new DateTime(2023, 12, 25); : Date définie pour le 25 décembre 2023.
    • TimeSpan duration = specificDate - now; : Différence entre deux dates.
    • string formattedDate = now.ToString("dd/MM/yyyy"); : Formatage en "jour/mois/année".
    • now.AddDays(5); : Ajoute 5 jours à la date actuelle.
  • LINQ et Collections :
    • List<int> numbers = new List<int> {1, 2, 3, 4}; : Liste d'entiers.
    • var evenNumbers = numbers.Where(n => n % 2 == 0).ToList(); : LINQ pour filtrer les nombres pairs.
    • numbers.ForEach(Console.WriteLine); : Parcourt et affiche chaque élément.
    • Dictionary<string, int> ages = new Dictionary<string, int> {{"Alice", 30}, {"Bob", 25}}; : Dictionnaire associant noms et âges.
    • var sum = numbers.Aggregate((a, b) => a + b); : Additionne tous les éléments d'une liste.
  • Asynchrone :
    • async Task LoadData() { await Task.Delay(1000); } : Méthode asynchrone avec délai de 1 seconde.
    • await Task.Run(() => DoWork()); : Exécute une tâche de manière asynchrone.
    • Task.WhenAll(task1, task2); : Attend que plusieurs tâches asynchrones soient terminées.

Java

  • Déclarations et Types :
    • int x = 10; : Variable entière.
    • String name = "John"; : Chaîne de caractères.
    • boolean isActive = true; : Variable booléenne.
    • double salary = 5000.50; : Double (nombre à virgule flottante).
  • Dates et Heures :
    • LocalDate date = LocalDate.now(); : Date actuelle.
    • LocalTime time = LocalTime.now(); : Heure actuelle.
    • LocalDateTime dateTime = LocalDateTime.of(2023, 12, 25, 15, 30); : Date et heure spécifiques.
    • Period period = Period.between(date, dateTime.toLocalDate()); : Différence en jours, mois, années.
    • dateTime.plusDays(5); : Ajoute 5 jours à la date actuelle.
  • Collections et Streams :
    • List<String> names = Arrays.asList("Alice", "Bob"); : Création d'une liste.
    • names.stream().filter(n -> n.startsWith("A")).forEach(System.out::println); : Filtre les noms commençant par "A" et les affiche.
    • int sum = Arrays.stream(new int[] {1, 2, 3}).sum(); : Somme de tous les éléments d'un tableau.
    • Map<String, Integer> map = new HashMap<>(); : Map (carte) associant des clés à des valeurs.

PHP

  • Déclarations et Types :
    • $x = 10; : Variable entière.
    • $name = "John"; : Chaîne de caractères.
    • $isActive = true; : Variable booléenne.
  • Gestion des Dates :
    • $now = new DateTime(); : Date et heure actuelles.
    • $specificDate = new DateTime("2023-12-25"); : Date spécifique.
    • $interval = $now->diff($specificDate); : Différence entre deux dates.
    • echo $now->format("d/m/Y H:i:s"); : Formate la date avec jour, mois, année et heure.
  • Connexion PDO :
    • $pdo = new PDO("mysql:host=localhost;dbname=test", "user", "password"); : Connexion sécurisée à une base de données.
    • $stmt = $pdo->prepare("SELECT * FROM users WHERE id = :id"); : Requête préparée pour éviter les injections SQL.
    • $stmt->execute(['id' => $user_id]); : Exécute la requête avec des paramètres.
    • $results = $stmt->fetchAll(PDO::FETCH_ASSOC); : Récupère les résultats en tableau associatif.

JavaScript

  • Déclarations et Types :
    • let x = 10; : Déclare une variable modifiable.
    • const name = "John"; : Déclare une constante (valeur immuable).
    • console.log("Hello World"); : Affiche un message dans la console pour le débogage.
  • Dates et Heures :
    • const now = new Date(); : Crée un objet de date pour l'heure actuelle.
    • const year = now.getFullYear(); : Obtient l'année actuelle.
    • const formattedDate = now.toLocaleDateString("fr-FR"); : Formate la date en français.
    • now.setDate(now.getDate() + 5); : Ajoute 5 jours à la date actuelle.
  • Fonctions :
    • function add(a, b) { return a + b; } : Fonction classique additionnant deux nombres.
    • const add = (a, b) => a + b; : Fonction fléchée (syntaxe concise) additionnant deux nombres.
    • (function() { console.log("Auto-exécutée"); })(); : Fonction auto-invoquée (IIFE).
  • DOM (Document Object Model) :
    • document.getElementById("myId"); : Sélectionne un élément par ID.
    • document.querySelector(".myClass"); : Sélectionne le premier élément avec une classe spécifique.
    • element.textContent = "Nouveau Texte"; : Change le texte d'un élément sélectionné.
    • element.style.color = "blue"; : Change la couleur de texte de l'élément.
  • Événements :
    • element.addEventListener("click", function() { alert("Clic détecté"); }); : Ajoute un écouteur d'événement de clic.
    • element.removeEventListener("click", handlerFunction); : Supprime un écouteur d'événement.
  • Promesses et Asynchrone :
    • fetch("url").then(response => response.json()).then(data => console.log(data)); : Utilise fetch pour récupérer des données asynchrones.
    • async function loadData() { const data = await fetch("url"); } : Fonction asynchrone utilisant await pour attendre les données.
