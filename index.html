SISR

Mémento SISR

  • Commandes Cisco :
    • show running-config : Affiche la configuration en cours.
    • enable : Passe en mode privilégié.
  • Commandes Linux :
    • ls : Liste les fichiers et répertoires.
    • cd : Change de répertoire.
  • Commandes Windows :
    • ipconfig : Affiche la configuration réseau de la machine.
    • ping : Teste la connectivité réseau avec une adresse IP.

SLAM

Mémento SLAM

C#

  • Déclarations : int x = 10;, string name = "John";, etc.
  • Dates et Heures : DateTime now = DateTime.Now;, TimeSpan duration = specificDate - now;.
  • LINQ : var evenNumbers = numbers.Where(n => n % 2 == 0).ToList();

Java

  • Déclarations : int x = 10;, String name = "John";, etc.
  • Dates et Heures : LocalDate date = LocalDate.now();.
  • Streams : names.stream().filter(n -> n.startsWith("A")).forEach(System.out::println);

PHP

  • Base : $x = 10;, $name = "John";.
  • Dates : $now = new DateTime();, $formattedDate = $now->format("d/m/Y");.
  • PDO : $pdo = new PDO("mysql:host=localhost;dbname=test", "user", "password");.

JavaScript

  • Déclarations : let x = 10;, const name = "John";.
  • Dates : const now = new Date();, const formattedDate = now.toLocaleDateString("fr-FR");.
  • DOM : document.getElementById("myId");, element.style.color = "blue";.
